## Commands
| Command | Description | Permission | Default |
| --- | --- | --- | --- |
| ```/snpc spawn``` | ```Spawn an npc``` | ```snpc.spawn``` | ```op``` |
| ```/snpc edit [args]``` | ```Edit the npc``` | ```snpc.edit``` | ```op``` |
| ```/snpc remove``` | ```Remove the npc``` | ```snpc.remove``` | ```op``` |
| ```/snpc list``` | ```See the npc list``` | ```snpc.list``` | ```op``` |
